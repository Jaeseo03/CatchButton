## 🔍 1단계 구현 시 어려웠던 점 및 해결 방법
- **버튼의 영역 이탈 문제**: 
  - **문제**: 단순히 랜덤 좌표를 생성할 경우 버튼의 일부가 폼(Form) 밖으로 나가는 현상이 발생
  - **해결**: 좌표 계산 시 폼의 전체 크기(`ClientSize`)에서 버튼의 크기(`Size`)를 뺀 범위 내에서만 무작위 값이 제한하여 화면 안에서만 이동하도록 구현
